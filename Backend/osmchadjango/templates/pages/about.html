{% trans 'About OSMCHA' %}

{% blocktrans %} OSMCHA is a tool to help identify OpenStreetMap changesets with a big number of changes or with suspect words like "import", "google", "waze", etc. Furthermore, it can be used as a peer revision system, any OSM user can verify the edits of another user. {% endblocktrans %}
